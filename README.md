# Malvinas, 1982 — Archivo Télam

Archivo fotográfico de la Guerra de Malvinas publicado por la agencia Télam. 255 fotografías organizadas en 10 galerías.

**https://martjanz.github.io/malvinas-telam/**

## Galerías

1. El desembarco
2. Operación prensa
3. Menéndez gobernador
4. El primer bombardeo
5. Vida cotidiana
6. Los soldados
7. Las trincheras
8. El Irizar
9. Las más conocidas
10. Los corresponsales

## Tecnología

Sitio estático — HTML, CSS y JavaScript vanilla. Sin dependencias externas ni frameworks. Publicado en GitHub Pages desde la carpeta `docs/`.

## Desarrollo

```bash
# Servir localmente
python3 -m http.server --directory docs
```
